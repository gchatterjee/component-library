gchatterjee component library

Gaurav Chatterjee's personal library of HTML5 custom components.

gchatterjee-slide

The gchatterjee-slide component should be used in conjunction with the gchatterjee-slide-container component. Together, the components create an element with quantized scrolling. Content inside the gchatterjee-slide is centered vertically and horizontally by default.
<gchatterjee-slide-container>
  <gchatterjee-slide><div>This is a slide.</div></gchatterjee-slide>
  <gchatterjee-slide><div>This is another slide.</div></gchatterjee-slide>
  <gchatterjee-slide><div>This is the last slide.</div></gchatterjee-slide>
</gchatterjee-slide-container>
