gchatterjee component library

Gaurav Chatterjee 's personal library of HTML5 custom components.

gchatterjee-slide

The gchatterjee-slide component should be used in conjunction with the gchatterjee-slide-container component. Together, the components create an element with quantized scrolling. Content inside the gchatterjee-slide is centered vertically and horizontally by default.
WARNING: Does not work cross browser, especially on mobile.
WARNING: No support for linking to sections.
<gchatterjee-slide-container>
  <gchatterjee-slide><div>This is a slide.</div></gchatterjee-slide>
  <gchatterjee-slide><div>This is another slide.</div></gchatterjee-slide>
  <gchatterjee-slide><div>This is the last slide.</div></gchatterjee-slide>
</gchatterjee-slide-container>

gchatterjee-circle-toc

The gchatterjee-circle-toc element creates an element to allow navigation through the page. gchatterjee-circle-toc supports the following attributes:
  • number specifies the active circle, starting at index 0.
  • of specifies the total number of circles
  • alignment determines the side on which tooltips appear. The value signifies the side of the parent element at which the gchatterjee-circle-toc is placed.
gchatterjee-circle-toc includes one slot named section-title. Elements slotted in this slot will be displayed as the tooltips, in the order in which they are slotted. An empty element signifies that a tooltip should not be displayed. Slotting no elements will result in no tooltips.
<gchatterjee-circle-toc number="1" of="4" alignment="right">
  <span slot="section-title" href="https://www.example.com">example.com</span>
  <span slot="section-title">no hyperlink</span>
  <span slot="section-title"></span>
  <span slot="section-title">fourth label</span>
</gchatterjee-circle-toc>
DEMO: Hover over the circes in the element to see the effect.

gchatterjee-link-bar

gchatterjee-link-bar allows a elements to be slotted to form a horizontal menu that becomes smaller when the div containing it is scrolled.
<gchatterjee-link-bar class="cold">
  <a slot="link"href="https://www.example.com/"> EXAMPLE.COM</a>
  <a slot="link" href="#">EMPTY LINK</a>
</gchatterjee-link-bar>
